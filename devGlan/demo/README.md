## Starting the project 

1. Go to https://start.spring.io/ 
2. Select a Maven Project in Java with Spring Boot version 2.2.4
3. Select the following 3 dependencies
	a) Spring Reactive Web
	b) Spring Security
	c) Spring Data Reactive Mongo
